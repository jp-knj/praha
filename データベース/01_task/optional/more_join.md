# More Join

#### List the films where the yr is 1962 [Show id, title]
```sql
SELECT id, title
 FROM movie
 WHERE yr=1962
```

#### Give year of 'Citizen Kane'
```sql
SELECT yr FROM movie WHERE title = 'Citizen Kane'
```
